# Spotify Clone 🎧

This is a basic Spotify UI clone built using HTML and CSS. The project includes:

- Navigation bar
- Music player layout
- Aligned cards and sections
- Partial responsiveness (mobile view WIP)
- Images stored in `assets/` folder
